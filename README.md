# Magic Stats App (WIP)

Front-end for Magic Stats API using React, Typescript, TailwindCSS, Headless UI, tanstack/query, tanstack/router.

## Started

```bash
yarn
# or `npm install`
```

---

## Develop

```bash
yarn dev
# or `npm run dev`
```

---

## Build

```bash
yarn build
# or `npm run build`
```
